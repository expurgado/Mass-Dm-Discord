## Mass DM Modes

This bot has **two Mass DM modes**:

### Mode 1 — Normal (Fast)

- Mode number: **1**
- Sends messages very fast
- Higher chance for the bot/account to get rate-limited or banned
- Recommended only for testing or short runs

### Mode 2 — Timed (Delay)

- Mode number: **2**
- Uses a time delay between each DM
- Slower but much safer
- Lower chance of rate limits or bans
- Recommended for safer and longer usage
